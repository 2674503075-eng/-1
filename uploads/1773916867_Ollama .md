### Ollama 核心指令

#### 一、对话界面指令（在模型聊天时使用）

| 指令                    | 功能说明                                                     |
| ----------------------- | ------------------------------------------------------------ |
| `/bye `或`/exit`        | 退出当前模型对话，返回命令行                                 |
| `/reset`                | 重置当前对话上下文（清空历史记录）                           |
| `/set system"内容"`     | 设置模型的系统提示词（列如：`/set system"你是专业的编程助手"`） |
| `/set temperature 数值` | 调整模型的随机性（0 = 严谨，1 = 创意，默认0.7）              |
| `/show`                 | 查看当前模型的配置（系统提示词，参数等）                     |
| `/model 模型名`         | 不退出界面，直接切换到指定模型（例如：`/mddel gemma:2b`）    |
| `/load 会话名`          | 加载之前保存的对话会话                                       |
| `/save 会话名`          | 保存当前对话到本地                                           |
| `/clear`                | 清空当前界面的聊天显示（仅视觉清空，上下文仍保留）           |

#### 二、全局命令行指令（直接在 CMD/PowerShell 执行）

| 指令                          | 功能说明                                                |
| ----------------------------- | ------------------------------------------------------- |
| `ollama ran 模型名[:标签]`    | 下载并运行指令模型 （标签可选，列如：`oll`）            |
| `ollama pull 模型名[:标签]`   | 仅下载模型到本地，不运行                                |
| `ollama rm 模型名[:标签]`     | 删除本地指定模型（例如：`ollama rm gemma:2b`）          |
| `ollama rm -a`                | 删除本地所有模型（谨慎使用）                            |
| `ollama cp 原模型名 新模型名` | 复制模型并重命名（例如：`ollama cp gemma:2b my-gemma`） |
| `ollama inspevt 模型名`       | 查看模型的详细元信息（大小，参数，创建时间等）          |
| `ollama serve`                | 启动Ollama后台服务（默认自动启动，无需手动执行）        |
| `ollama version`              | 查看当前Ollama的版本号                                  |

#### 三、高级操作指令

| 指令                                          | 功能说明                                                   |
| --------------------------------------------- | ---------------------------------------------------------- |
| `ollama create 自定义模型名 -f Modelfile路径` | 基于Modelfile创建自定义模型<br />（例如微调，组合模型）    |
| `ollama push 模型名`                          | 将本地模型推送到Ollama云端库<br />（需登录账户）           |
| `ollama pull 用户名/模型名`                   | 拉取他人分享的自定义模型<br />（例如：`ollama pull user`） |
| `ollama run 模型名 --verbose`                 | 运行模型时显示详细日志（用于调试）                         |
